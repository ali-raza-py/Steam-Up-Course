# 🚀 Steam-Up-Course

**My complete coursework, projects, and daily class notes from the Steam-Up programming course.**

This repository documents my full learning journey — **51 classes**, **150+ Python files**, and **10,000+ lines of code** — progressing from Python basics all the way to Object-Oriented Programming and Data Structures & Algorithms (DSA). Each numbered folder is one class session, in the order the topics were taught, with a `README.md` inside explaining exactly what that session covered.

---

## 📌 Why this repo exists

- Keep a transparent, complete record of everything covered in the course — including the class I was absent for and the exam/paper sessions
- Practice writing and committing real, working code for every lesson instead of just watching along
- Give **other students on the same course** (or anyone learning Python from scratch) a day-by-day reference to catch up, revise, or compare progress
- Collect all homework and assignments in one place

---

## 🧭 Learning Progression — 5 Phases

| Phase | Classes | Focus |
|---|---|---|
| **Phase 1: Foundations** | 1–10 | Variables, conditionals, loops, functions, pattern printing, basic math algorithms |
| **Phase 2: Intermediate** | 11–25 | Tuples, dictionaries, exception handling, GUI (Tkinter), NumPy fundamentals |
| **Phase 3: Data Analysis & Visualization** | 26–32 | NumPy (advanced), Pandas, Matplotlib, Seaborn, Dash |
| **Phase 4: Object-Oriented Programming** | 33–43 | Classes, inheritance, encapsulation, abstraction, OOP projects |
| **Phase 5: Advanced DSA** | 44–51 | Stacks, queues, binary search trees, algorithm-driven applications |

## ✅ How Others Can Benefit From This Repo

- **Fellow course students**: jump into any class's `README.md` to see exactly what was covered, compare solutions, or catch up on a missed session.
- **Self-learners**: follow the numbered folders in order as an informal Python → DSA curriculum path (see the Phase table above).
- **Interview prep**: go straight to `SORTING ALG` and the `DSA` folders inside Classes 44, 46, 47, 49–51 for algorithm practice (stacks, queues, BST, sorting).
- **Instructors/mentors**: use this as a template for day-by-day student progress tracking with per-session documentation.

## 📖 How to Navigate

1. Start at `1ST CLASS` and move sequentially for the full learning path.
2. Jump to `SORTING ALG` or the `DSA` sub-folders (Classes 44/46/47) for algorithm-only practice.
3. Check `Basic projects` for small, self-contained apps that combine multiple concepts.
4. `H.W` and `Assighment Week 2` show concepts applied under assignment constraints.
5. Every class folder has its own `README.md` — open it before the code for context.
